Preserve Files when Deregistering

Run the book so that you do not lose data when deregistering.

Download

Download preserve.sh

Instructions

  1. Download
  2. Move preserve.sh to kindle's documents
  3. Open kindle and run the booklet
  4. Deregister
  5. Copy all documents onto pc
  6. Then back onto kindle

Notes

The kindle expects the documents to be empty, however we made it not be empty which is why we move the files back thankfully we don't have to reinstall anything now
