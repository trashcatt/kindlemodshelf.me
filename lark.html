LARK

LARK is a Libre Audiobook Player for Kindle. It uses KinAMP's audio engine and is optimized for e-book readers.

Download

  • Download LARK from GitHub

Features

  • Support for AAC encoded M4B audiobooks
  • Uses KinAMP's audio engine and FAAD2 decoder library
  • Optimized for e-book readers: minimum screen refreshes, backlight management
  • Listening history
  • Scriptlet and KUAL launcher included

Installation

  1. Download the latest release from GitHub and extract the archive.
  2. Put lark/LARK in your root directory.
  3. Put lark/extensions/lark folder in your extensions folder on your Kindle.
  4. Put lark/documents/lark.sh script in your Kindle documents folder.
  5. Start it from the library using the provided scriptlet or from KUAL.
